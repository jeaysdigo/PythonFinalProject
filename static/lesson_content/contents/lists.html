Python Lists

List in Python

Lists in Python serve as a versatile way to store multiple items in a single variable.

Creating Lists

Example:

 # Create a List
 thislist = ["apple", "banana", "cherry"]
 print(thislist)

List Characteristics

  • Ordered: Lists maintain a defined order, and the order remains unchanged unless explicitly modified. New items are typically added at the end of the list.
  • Changeable: Lists are mutable, allowing for the addition, removal, and alteration of items after creation.
  • Allow Duplicates: Lists can con>tain items with the same value.

List Length

To ascertain the number of items in a list, the len() function is used:

Example:

 # Print the number of items in the list
 thislist = ["apple", "banana", "cherry"]
 print(len(thislist))

List Items - Data Types

Lists can accommodate items of various data types, including strings, integers, and booleans:

Example:

 # String, int, and boolean data types
 list1 = ["apple", "banana", "cherry"]
 list2 = [1, 5, 7, 9, 3]
 list3 = [True, False, False]
    
 # A list with mixed data types
 list4 = ["abc", 34, True, 40, "male"]

Type of Lists

From Python's perspective, lists are objects with the data type 'list':

Example:

 # Data type of a list
 mylist = ["apple", "banana", "cherry"]
 print(type(mylist))

Using the list() Constructor

Alternatively, the list() constructor can be employed to create a new list:

Example:

 # Using the list() constructor to make a list
 # Note the double round-brackets 
 thislist = list(("apple", "banana", "cherry")) 
 print(thislist)
