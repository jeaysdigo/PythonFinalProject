Python Tuples

Tuple in Python

Tuples in Python serve as a means to store multiple items in a single variable.

Creating Tuples

Lists can accommodate items of various data types, including strings, integers, and booleans:

Example:

 # Defining a Tuple
 fruit_tuple = ("apple", "banana", "cherry")
 print(fruit_tuple)

List Characteristics

  • Ordered: Tuples maintain a defined order, and this order remains fixed.
  • Unchangeable: Tuples are immutable, meaning that their items cannot be altered, added, or removed after creation.
  • Allow Duplicates: Tuples can contain items with the same value.

Tuple Length

To determine the number of items in a tuple, the len() function is used:

Example:

 # Display the count of elements in the tuple
 animal_tuple = ("lion", "elephant")
 print(len(animal_tuple))

Creating Tuple With One Item

To create a tuple with only one item, a comma must be added after the item; otherwise, Python will not recognize it as a tuple:

Example:

 # One item tuple, remember the comma
 thistuple = ("apple",)
 print(type(thistuple))

 # NOT a tuple
 thistuple = ("apple")
 print(type(thistuple))

Tuple Items - Data Types

Tuple items can be of any data type, including strings, integers, and booleans:

Example:

 # String, int, and boolean data types
 tuple1 = ("apple", "banana")
 tuple2 = (1, 5, 7)
 tuple3 = (True, False)

 # A tuple with mixed data types
 tuple4 = ("abc", 34, True)

Type of Tuples

From Python's perspective, tuples are objects with the data type 'tuple':

Example:

 # Data type of a tuple
 # Note the double round-brackets 
 mytuple = ("apple", "banana")
 print(type(mytuple))

Using the tuple() Constructor

Alternatively, the tuple() constructor can be employed to create a new tuple:

Example:

 # Data type of a tuple
 thistuple = tuple(("apple", "banana", "cherry")) 
 print(thistuple)
