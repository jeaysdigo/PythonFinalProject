Python Numbers

Python Numeric Types

Python provides three main numeric types:

  • int (Integer)
  • float (Floating-point)
  • complex (Complex)

Example: Declaring Numeric Variables

 # Numeric types in Python
 x = 1    # Integer
 y = 2.8  # Float
z = 1j   # Complex

 # Verify the type of each variable
 print(type(x))
 print(type(y))
 print(type(z))

Integer (int)

Integers are whole numbers, positive or negative, without decimals and of unlimited length.

Example: Integer Variables

 # Integers
 x = 1
 y = 35656222554887711
 z = -3255522

 print(type(x))
 print(type(y))
 print(type(z))

Float (float)

Floats are numbers with one or more decimals, including scientific notation with an "e" to indicate the power of 10.

Example: Float Variables

 # Float
 x = 1.10
 y = 1.0
 z = -35.59

 print(type(x))
 print(type(y))
 print(type(z))

 # Scientific Notation
 x = 35e3
 y = 12E4
 z = -87.7e100

 print(type(x))
 print(type(y))
 print(type(z))

Complex (complex)

Complex numbers are written with a "j" as the imaginary part.

Example: Complex Variables

 # Complex numbers
 x = 3+5j
 y = 5j
 z = -5j

 print(type(x))
 print(type(y))
 print(type(z))

Type Conversion

You can convert between numeric types using the int(), float(), and complex() methods.

Example: Type Conversion

 # Type conversion
 x = 1    # int
 y = 2.8  # float
 z = 1j   # complex

 # Convert from int to float
 a = float(x)

 # Convert from float to int
 b = int(y)

 # Convert from int to complex 
 c = complex(x)

 print(a)
 print(b)
