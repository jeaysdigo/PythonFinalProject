Python Conditional Statements

Introduction to Python Conditions and If Statements

Python incorporates familiar logical conditions from mathematics:

  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b

These conditions are applied in "if statements" and loops in various ways.

An "if statement" uses the if keyword.

Example:

Example of an if statement:

 number_a = 33
 number_b = 200
 if number_b > number_a:
 print("number_b is greater than number_a")

In this instance, two variables, number_a and number_b, are used in an if statement to test whether number_b is greater than number_a. Given that number_a is 33 and number_b is 200, the condition is true, so "number_b is greater than number_a" is printed to the screen.

Elif: The Conditional Else If

The elif keyword is Python's way of saying "if the previous conditions were not true, then test this condition".

Example:

Example of an elif statement:

 number_a = 33
 number_b = 33
 if number_b > number_a:
    print("number_b is greater than number_a")
 elif number_a == number_b:
    print("number_a and number_b are equal")

In this example, number_a is equal to number_b, so the first condition is false. However, the elif condition is true, leading to the output "number_a and number_b are equal".

The Else Condition

The else keyword captures anything that isn't covered by the preceding conditions.

Example:

Example of an else statement:

 number_a = 200
 number_b = 33
 if number_b > number_a:
    print("number_b is greater than number_a")
 elif number_a == number_b:
    print("number_a and number_b are equal")
 else:
    print("number_a is greater than number_b")

In this example, number_a is greater than number_b, so the first condition is false. Additionally, the elif condition is false. As a result, the else condition is executed, printing "number_a is greater than number_b" to the screen.

Short Hand If

If you have only one statement to execute, you can place it on the same line as the if statement.

Example:

One-line if statement:

 if number_a > number_b: print("number_a is greater than number_b")

Short Hand If ... Else

If you have only one statement for both if and else, you can consolidate them onto the same line:

Example:

One-line if-else statement:

 number_a = 2
 number_b = 330
 print("A") if number_a > number_b else print("B")
