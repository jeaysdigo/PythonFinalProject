Python Basics

Understanding Python Indentation

Indentation in Python refers to the spaces or tabs at the beginning of a line of code. While in many other programming languages, indentation is primarily for readability, in Python, it serves a crucial role in determining code structure and execution.

Python uses indentation to signify blocks of code, such as loops, conditionals, and functions. The way you indent your code directly influences its behavior, making indentation a fundamental aspect of Python syntax.

Let's explore a simple example to illustrate the significance of indentation:

Example:

 if 5 + 5 == 10:
    print("Five plus five equals ten!")
 else:
    print("This statement is outside the 'if' block.")

Python will give you an error if you skip the indentation:

Example:

 if 3 < 5:
 print("Three is less than five!")

Your First Python Program

The print() function in Python is used to display or output text or variables to the console or terminal.

Example:

 print("Hello, World!")

Python Comments

  • Comments in Python are used to provide explanations for code, improving readability.
  • They help make the code more understandable for others (or even for yourself in the future).
  • Comments can also be used to temporarily disable or "comment out" code during testing.

Creating a Comment

In Python, comments start with the # symbol, and the interpreter ignores them:

Example:

# This is a single-line comment
print("Hello, World!")

Multiline Comment

Although Python does not have a specific syntax for multiline comments, you can use the # symbol for each line:

Example:

# This is a multiline comment
# written in
# more than just one line
print("Hello, World!")

Scope of Variables

In Python, the scope of a variable defines the region in the code where the variable can be accessed or modified. Understanding variable scope is crucial for writing efficient and bug-free code.

There are two main types of variable scope in Python:

  • Global Scope: Variables defined outside of any function or block have a global scope. They can be accessed from anywhere in the code.
  • Local Scope: Variables defined inside a function or block have a local scope. They are only accessible within that specific function or block.

Example:

 global_variable = "I am global"

 def example_function():
     local_variable = "I am local"
     print(global_variable)
     print(local_variable)

 example_function()
 print(global_variable)
 # Attempting to print local_variable here would result in an error

It's essential to be mindful of variable scope to avoid unexpected behavior in your Python programs.
