Python String Operations

String Basics

Strings in Python are versatile and can be enclosed in single or double quotation marks.

'hello' and "hello" are interchangeable.

Display a string literal using the print() function:

Example:

 print("Greetings")
 print('Greetings')

Variable Assignment with Strings

Assigning a string to a variable is accomplished with the variable name, an equal sign, and the string value:

Example:

 greeting = "Hello"
 print(greeting)

Multiline Strings in Action

Use triple quotes for multiline string assignment:

Both triple single and triple double quotes are valid:

Example:

 message = '''Lorem ipsum,
 dolor sit amet,
 consectetur adipiscing elit.'''
 print(message)

Strings as Arrays

Strings in Python are arrays of bytes representing Unicode characters.

While Python lacks a specific character data type, a single character is a string with a length of 1.

Access individual elements using square brackets:

Example:

Retrieve the character at position 1 (remember, the first character is at position 0):

 word = "Hello, World!"
 print(word[1])

Iterating Through Strings

As strings are arrays, iterate through characters with a for loop.

Example:

Iterate through the letters in the word "banana":

 for letter in "banana":
 print(letter)

String Length Check

Determine the length of a string using the len() function.

Example:

The len() function provides the string's length:

 phrase = "Hello, World!"
 print(len(phrase))

String Presence Check

Verify if a specific phrase or character exists in a string using the in keyword.

Example:

Check if "World" is present in the following text:

  text_sample = "Hello, World!"
 print("World" in text_sample)

Negation Check

Determine if a particular phrase or character is NOT present in a string using the not in keyword.

Example:

Check if "expensive" is NOT present in the following text:

 quote = "The best things in life are free!"
 print("expensive" not in quote)
