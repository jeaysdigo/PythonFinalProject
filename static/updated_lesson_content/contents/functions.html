Understanding Python Functions

Python Functions Overview

A function in Python is a block of code that executes only when called, and it may return data as a result.

Defining a Function

In Python, use the def keyword to define a function:

Example:

def my_function():
    print("Hello from a function")

Invoking a Function

To call a function, use its name followed by parentheses:

Example:

def my_function():
     print("Hello from a function")

 my_function()

Function Arguments

Information can be passed to functions as arguments.

Arguments are specified inside parentheses after the function name. Multiple arguments are separated by commas.

In the following example, the function takes one argument (fname). When called, it prints the full name using the provided first name:

Example:

def my_function(fname):
    print(fname + " Refsnes")

    my_function("Emil")
    my_function("Tobias")
     my_function("Linus")

Returning Values

To make a function return a value, use the return statement:

Example:

def my_function(x):
    return 5 * x

    print(my_function(3))
    print(my_function(5))
    print(my_function(9))
