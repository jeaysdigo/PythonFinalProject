Python Booleans

Python Booleans

Booleans in Python represent binary values of either True or False.

Boolean Values

You can compare values using expressions, and Python will return either True or False:

Example:

 print(10 > 9)
 print(10 == 9)
 print(10 < 9)

In conditional statements like if statements, Python returns True or False:

Example:

 a = 200
 b = 33
    
 if b > a:
    print("b is greater than a")
 else:
    print("b is not greater than a")

Evaluating Values and Variables

The bool() function allows you to assess any value and obtain a True or False result:

Example:

 print(bool("Hello"))
 print(bool(15))
    
 x = "Hello"
 y = 15
    
 print(bool(x))
 print(bool(y))

Truthiness in Values

Most values are considered True if they have content. Strings, numbers (except 0), and non-empty lists, tuples, sets, and dictionaries are True. Empty values like "", 0, (), [], {} and the value None evaluate to False:

Example:

# True values
 bool("abc")
 bool(123)
 bool(["apple", "cherry", "banana"])
    
 # False values
 bool(False)
 bool(None)
 bool(0)
 bool("")
 bool(())
 bool([])
 bool({})

Functions Returning Booleans

Functions can be designed to return Boolean values. For instance:

Example:

# Function returning True 
 def myFunction():
    return True
    
 print(myFunction())
    
 # Executing code based on the Boolean result 
 def myFunction():
    return True
    
 if myFunction():
    print("YES!")
 else:
    print("NO!")

Built-in Functions Returning Booleans

Python has built-in functions like isinstance() that return Boolean values. For example:

Example:

 x = 200
 print(isinstance(x, int))
