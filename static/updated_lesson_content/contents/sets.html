Python Sets

Sets in Python

Sets are a powerful and flexible data type that allows you to store collections of unique items in a single variable.

Creating a Set

Sets in Python are denoted by curly brackets. Let's create a set and print its content:

Example:

 my_set = {"apple", "banana", "cherry"}
 print(my_set)

List Characteristics

  • Unordered: Sets are unordered, meaning the elements in a set do not have a defined order.
  • Unchangeable: Sets exhibit immutability, signifying that the individual elements within a set remain unalterable.
  • Unique: Sets only allow unique items; they cannot contain multiple elements with the same value.

Uniqueness and Order

Sets do not allow duplicate values, and the order of elements is not guaranteed.

Example:

 duplicate_set = {"apple", "banana", "cherry", "apple"}
 print(duplicate_set)  
 # Duplicate values are automatically ignored

Set Operations

Sets support various operations such as union, intersection, and difference. These operations make it easy to compare and manipulate sets.

Example:

 set1 = {1, 2, 3}
 set2 = {3, 4, 5}
    
 union_set = set1 | set2 
 intersection_set = set1 & set2  
 difference_set = set1 - set2 
    
 print(union_set)
 print(intersection_set)
 print(difference_set)

Set Methods

Python provides several built-in methods for sets. These include add(), remove(), and discard(). Let's explore these methods:

Example:

 fruits = {"apple", "banana", "cherry"}
 fruits.add("orange")
 fruits.remove("banana")
 print(fruits)

Length of a Set

You can determine the number of items in a set using the len() function:

Example:

 my_set = {"apple", "banana"}
 print(len(my_set))

Sets with Different Data Types

Sets can contain elements of various data types, providing flexibility in data representation:

Example:

 mixed_set = {"abc", 34, True, 40, "male"}
