Python Casting

Declare Variable Type

In certain situations, it may be necessary to explicitly specify the type of a variable. Python allows this through casting, where you use constructor functions to create variables of a particular type.

Casting in Python involves using the following constructor functions:

  • int() : Constructs an integer from an integer literal, a float literal (by removing all decimals), or a string literal (if the string represents a whole number).
  • float() : Constructs a float from an integer literal, a float literal, or a string literal (if the string represents a float or an integer).
  • str() : Constructs a string from various data types, including strings, integer literals, and float literals.

Example:

Integers:

 x = int(1)    # x will be 1
 y = int(2.8)  # y will be 2
 z = int("3")  # z will be 3

Example:

Floats:

 x = float(1)     # x will be 1.0
 y = float(2.8)   # y will be 2.8
 z = float("3")   # z will be 3.0
 w = float("4.2") # w will be 4.2

Example:

Strings:

 x = str("s1") # x will be 's1'
 y = str(2)    # y will be '2'
 z = str(3.0)  # z will be '3.0'
