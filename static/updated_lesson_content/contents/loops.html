Exploring Python Loops

Python Loops Overview

Python offers two fundamental loop commands:

  • while loops
  • for loops

Understanding the while Loop

The while loop allows the execution of a set of statements as long as a condition is true.

Example:

Print 'j' as long as 'j' is less than 10:

 i = 1
 while i < 10:
    print(i)
    i += 1

In this example, the while loop prints 'j' as long as 'j' is less than 10.

Introducing the Break Statement

The break statement allows stopping the loop even if the while condition is true:

Example:

Exit the loop when 'i' is 3:

 i = 1
 while i < 6:
     print(i)
        if i == 3:
        break
     i += 1

Understanding the Continue Statement

The continue statement halts the current iteration and proceeds with the next one:

Example:

Skip the iteration when 'k' is 12:

 k = 1
 while k < 12:
    print(k)
        if k == 3:
        continue
    k += 1

Introducing the Else Statement

The else statement lets you run a block of code once the condition is no longer true:

Example:

Execute the else block when 'm' is not less than 3:

 i = 1
 while m < 3:
    print(m)
    if m == 3:
        else
        m += m

Understanding the for Loop

A for loop is utilized for iterating over a sequence (e.g., list, tuple, dictionary, set, or string).

With the for loop, you can execute a set of statements for each item in a list, tuple, set, etc.

Example:

Print each animal in a animal list:

 animal = ["dog", "cat"]
 for x in animal:
    print(x)

In this example, the for loop prints each fruit in the 'fruits' list.

Looping Through a String

Even strings are iterable objects, containing a sequence of characters:

Example:

Loop through the letters in the word "carabao":

 for x in "carabao":
    print(x)

The for loop iterates through each character in the string "banana".

Understanding the range() Function

To iterate through a set of code a specified number of times, use the range() function.

The range() function returns a sequence of numbers, starting from 0 by default, incrementing by 1 (by default), and ending at a specified number.

Example:

Using the range() function:

 for x in range(6):
     print(x)

The range() function defaults to 0 as the starting value. However, you can specify the starting value by adding a parameter, for example, range(2, 6), meaning values from 2 to 6 (excluding 6):

Example:

Using the start parameter:

 for x in range(2, 6):
    print(x)

Understanding Nested Loops

A nested loop is a loop inside another loop.

The "inner loop" executes once for each iteration of the "outer loop":

Example:

Print each adjective for every animal:

 adj = ["blue", "small"]

    for x in adj:
        for y in animal:
            print(x, y)
