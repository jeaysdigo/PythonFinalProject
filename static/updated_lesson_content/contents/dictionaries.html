Python Dictionaries

Dictionaries in Python

A dictionary is a collection of key-value pairs, where each key must be unique. The key acts as an identifier for the associated value.

Creating a Dictionary:

Example:

# Example dictionary 
 my_dict = {
    "name": "John",
    "age": 25,
    "city": "New York"
 }

 print(my_dict)

Accessing Values:

To access the value of a specific key, use square brackets [] :

Example:

 # Accessing value using key
 print(my_dict["name"])

Modifying Values:

Dictionaries are mutable, allowing you to modify the values associated with keys:

Example:

 # Modifying value
 my_dict["age"] = 26
 print(my_dict)

Adding a New Key-Value Pair:

You can add a new key-value pair to a dictionary:

Example:

 # Adding a new key-value pair
 my_dict["occupation"] = "Engineer"
 print(my_dict)

Removing a Key-Value Pair:

To remove a key-value pair, you can use the del keyword:

Example:

 # Removing a key-value pair
 del my_dict["city"]
 print(my_dict)

Dictionary Methods:

Dictionaries come with various built-in methods. Some commonly used ones include:

  • keys(): Returns a list of all keys.
  • values() Returns a list of all values.
  • items() Returns a list of key-value pairs as tuples.

Example:

 # Using dictionary methods
 keys = my_dict.keys()
 values = my_dict.values()
 items = my_dict.items()

 print("Keys:", keys)
 print("Values:", values)
 print("Items:", items)

Dictionary Comprehension:

Similar to list comprehensions, Python allows dictionary comprehensions for concise dictionary creation:

Example:

 # Dictionary comprehension
 squares = {x: x*x for x in range(1, 6)}
 print(squares)
