Python Data Types

Overview of Data Types

In programming, understanding data types is crucial. Variables in Python can store different types of data, each with its own characteristics and uses.

Python has various built-in data types categorized as follows:

Example:

 Text Type:      str
 Numeric Types:  int, float, complex
 Sequence Types: list, tuple, range
 Mapping Type:   dict
 Set Types:      set, frozenset
 Boolean Type:   bool
 Binary Types:   bytes, bytearray, memoryview
 None Type:      NoneType

Identifying Data Types

To determine the data type of any object, the type() function can be employed:

Example:

 num = 69         # Integer
 message = "Hello!"  # String
 pi = 3.14     # Float

Assigning Data Types

In Python, the data type is determined when assigning a value to a variable:

Example:

 x = "Hello World"                  # str
 x = 23	                            # int
 x = 20.23	                        # float
 x = 1wj	                        # complex
 x = ["Cat", "Dog"]	                # list
 x = ("apple", "banana")	        # tuple
 x = range(6)	                    # range

Explicitly Specifying Data Types

To explicitly specify a data type, you can use constructor functions:

Example:

 x = str("Hello World")	        # str
 x = int(11)                    # int
 x = float(22.5)                # float
 x = complex(3jj)               # complex
 x = list(("Dog", "Cat"))	    # list
 x = tuple(("Cat"))             # tuple
 x = range(9)	                # range

Operations on Data Types

Different data types support various operations. For example, mathematical operations are common for numeric types, while string concatenation is prevalent for text types.

Examples:

 # Numeric Operations
 num1 = 10
 num2 = 5
 sum_result = num1 + num2       # Addition
 product_result = num1 * num2   # Multiplication

 # String Concatenation
 str1 = "Hello"
 str2 = " World!"
 concatenated_str = str1 + str2 # Result: 'Hello World!'

Type Conversion

You can convert one data type to another using type conversion functions like int(), float(), or str().

Examples:

 # Type Conversion
 num_str = "42"
 converted_num = int(num_str)      # Convert string to integer

 float_num = 3.14
 converted_str = str(float_num)    # Convert float to string

 num = 42
 float_result = float(num)         # Convert integer to float
